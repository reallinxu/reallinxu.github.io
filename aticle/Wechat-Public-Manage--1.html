[Wechat-Public-Manage]1.前端VUE初始化

发表于 2019-08-20 | 分类于 Wechat-Public-Manage |
字数统计: 65 字 | 阅读时长 ≈ 1 分钟

准备

  1. 安装nodejs，选择LTS版本：下载
  2. 使用淘宝镜像
    npm install -g cnpm --registry=https://registry.npm.taobao.org

开始

  1. 安装vue-cli
    cnpm install vue-cli -g //全局安装 vue-cli
  2. 创建项目
    vue init webpack wechat-public-manage-front
  3. 运行
    cnpm install
    cnpm run dev
  4. 查看结果
