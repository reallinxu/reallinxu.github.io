**第一步：建立git仓库**
cd到你的本地项目根目录下，执行git命令

    git init
**第二步：将项目的所有文件添加到仓库中**  
  
    git add .  
    如果想添加某个特定的文件，只需把.换成特定的文件名即可

**第三步：将add的文件commit到仓库**  

    git commit -m "注释语句"  

**第四步：将commit的文件push到仓库**  

    git push -u origin dev
